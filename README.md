# Анекдот-бот на основе ChatGPT

Консольное приложение для рассказа анекдотов с использованием Bothub API.

##  Возможности

- Рассказывает анекдоты на любые указанные темы, либо же случайный, если тема не указана.
- Хранит историю последних 5 диалогов
- Поддерживает различные команды
- Показывает статистику сессии
